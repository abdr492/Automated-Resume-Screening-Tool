# 🚀 Project Overview

Automated Resume Screening Tool is a Python-based NLP and Machine Learning system that automatically analyzes resumes, compares them with job descriptions, calculates matching scores, and ranks candidates based on relevance.

The project helps automate the resume shortlisting process for recruiters and HR teams.

This project demonstrates real-world applications of:
- NLP (Natural Language Processing)
- Resume Parsing
- Text Similarity Analysis
- Machine Learning
- HR Tech Automation
- Candidate Ranking Systems
